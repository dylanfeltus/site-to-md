# sitetomd.com Landing Page

Landing page for [site-to-md](https://github.com/dylanfeltus/site-to-md).

Built with:
- Next.js 15 + App Router
- Tailwind CSS v4
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deploys automatically to Vercel on push to main.

```bash
npm run build
```

Static output goes to `./out`
